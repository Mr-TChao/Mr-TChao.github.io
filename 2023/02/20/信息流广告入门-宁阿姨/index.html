信息流广告入门--宁阿姨

基础篇

第一章: 信息流广告基本介绍

1.1 信息流广告是什么

1.1.1 信息流广告的定义

信息流(feed)广告,是在内容流中穿插展现的广告, 常出现在社交网络 资讯媒体和视频平台,
以文字、图片、视频或他们相结合的方式展现。它主要出现在移动端，现对于传统的网络广告， 内容更加灵活、有趣，受用户欢迎程度更高。

说明
用户自发发布的内容叫做内容，给平台带来明确收益的商业推广叫做广告

信息流广告的形式和用户发布的状态很相似，不会让用户觉得太突兀。

1.1.2 信息流广告的特点

信息流广告的关键词： 原生 和 效果。
信息流广告的一大优势是原生，即广告和环境内容在形式上高度相似。
相比传统的网站悬浮广告和弹窗广告，信息流广告的用户体验更好。信息流广告因为形式上“长得很像”内容，
容易在不经意间“溜”进用户的视线，降低被用户直接过滤的可能性，所以更容易被用户接受。

信息流广告对比传统广告的优势
传统广告被吐槽的一个点就是“所有人看到的都是一样的广告，不管你有没有需求。”
信息流广告的一个核心在于通过对技术的应用，尽量避免将广告展现给不需要的人。通过推荐算法确保广告精准匹配用户，对不同的用户展现不同的广告，这就是所谓的千人千面，从而实现广告和用户的精准匹配。
与传统的地铁、公交广告的效果难以估算不同，信息流广告的投放都在线上，广告投放效果能够量化。
信息流广告的原生性及对目标用户的精准匹配，提高了广告投放的效率。

信息流广告精准匹配背后隐藏的技术
用户的每一次上网行为都会留下痕迹（点击、搜索、浏览等），媒体通过一定的以技术手段对用户行为数据进行收集分析，从而形成对用户偏好的判断。再结合具体的环境（时间、网络环境和位置信息等），向用户推荐其可能感兴趣的广告，从而提升广告和用户的匹配度，促进用户转化。

人群特征 + 环境特征 = 广告推送

效果广告和品牌广告
希望借助媒体的力量来快速接触大量用户，以达到宣传品牌形象、提升中长期购买率与利润空间的目的。这种目的广告称为品牌广告。
希望利用广告手段马上带来大量的购买或其他转化行为，这种目的的广告称为直接效果广告，简称效果广告。

1.2 信息流广告的行业发展

1.2.1 网络广告的常见形式

网络广告是指广告主基于互联网所投放的广告。
互联网广告的设计思路就是在用户关注度高的地方拦截用户的注意力，让用户顺便或者不得不看到广告。

  1. 开屏广告（闪屏广告）
    开屏广告指在APP启动时占据屏幕大部分或全部面积的广告。
    特点：开屏广告的覆盖面最广，所以价格一般相对较高，点击率比其他广告高一些。

  2. banner广告
    banner广告就是横幅广告，以通栏或者矩形框的形式出现在页面中，一般展现在用户停留较久或者访问频繁的页面上。
    特点：网络广告最早的展现形式，但是点击率相对较低。

  3. 插屏广告
    插屏广告一般出现在APP暂停或者场景切换的时候，以半屏、全屏的方式弹出。
    常见场景有视频的前贴、暂停、后贴、游戏暂停、过关和图书翻页等。
    特点：都出现在一些比较“特殊”的场景里，所以能减少广告对APP固定界面的占用，所以游戏和视频等产品多采用这种广告形式。
    注意：有的地方将贴片广告和插屏广告算作两类广告。贴片广告主要跟随视频的播放出现，包含视频播放之前的前贴片广告和播放之后的后贴片广告。

  4. 激励广告
    激励广告多为视频形式，通过设置奖励来吸引用户主动观看广告。
    激励广告多出现在游戏里，给用户的奖励通常是人物复活的机会、一些道具或者特权等。
    特点：

    • 保证广告的播放时长。用户一旦选择看广告，就必须播放完毕才能领取奖励。
    • 用对广告的接受程度相对较高。因为用户是主动选择播放广告，并且播放完毕可以获得直接收益，所以从一定程度上确保了广告效果。

  5. 搜索引擎广告
    搜索引擎广告是通过用户搜索关键词而触发的广告。搜索引擎广告是典型的原生广告，广告的样式和搜索出来的词条内容非常相似，并且广告的内容和用户的搜索意图相关，形势和内容上都很“原生”，因此转化效果较好。

  6. 信息流广告
    信息流广告和搜索引擎广告是原生广告的两个类型代表，两者在形势和内容上很相似。
    不同的是，信息流广告是媒体主动推荐给用户的广告，可以出现在任意内容的中间；而搜索引擎广告是用户搜索关键词之后才能展现的，局限性很强，所以信息流广告的曝光机会更多。

信息流广告的优势

  • 不像开屏广告和视频前贴片广告那样具有强制性，用户不喜欢就可以划过去。
  • 和banner广告相比，与用户的相关性更强，广告的展示基于用户兴趣，千人千面。
  • 非常适合移动端的碎片化阅读。
  • 无论是社交产品、新闻产品还是视频产品，每条内容之间的相关性都很弱，并不存在严格的上下文联系，所以在信息流中插入一条广告也不会特别的干扰用户。

1.2.2 信息流广告的规模

信息流广告是流量变现的一种高效方式

1.2.3 主要企业

  1. 字节跳动
  2. 腾讯
    腾讯的优势是基于社交网络下丰富的用户行为（发动态、好友关系、支付等），对用户的了解比较精确，有利于实现广告和用户的精准匹配。
  3. 百度
  4. 快手

1.3 信息流广告在产业链中的主要角色

整个产业链主要由三方组成，分别是媒体、广告主和代理商。

媒体
流量提供商，比如百度，腾讯等。用户在哪，广告就在哪，每一个大流量主都是新时代的“媒体”。

广告主
流量需求方，希望通过广告来获取用户或者促进销售，是这个产业链中的甲方。
信息流广告发展快速的原因：广告效果被广告主认可，广告主才是买单的那一方。

代理商
提供优化服务的一方，帮助广告主提升广告投放效果。

媒体的诉求
将广告卖出去，赚足钱。
主要工作：通过技术提升广告效果，让广告主买单。信息流广告是效果广告，让广告精准匹配目标用户，这个匹配的过程靠的是媒体的算法。算法能力是媒体的核心能力。

广告主的诉求
核心诉求： 获客。通过广告增加曝光，几乎是每一个产品必不可少的营销方式。

代理商的诉求
代理商最重要的就是优化能力， 能够帮助广告主在媒体上投放广告，优化广告效果。（代理商是不一定存在的）

流量（用户）是整个产业的核心。媒体拥有流量，卖广告；广告主需要流量，买广告；代理商服务买卖的过程。

1.4 一条广告是怎么出现的

  1. 广告投放首先要进行人群筛选，这个过程称之为定向。
  2. 广告准需要准备广告投放的物料（手机看看到的图片和广告词），称之为广告创意。
    创意是直接给用户看的，需要考虑用什么样的创意能够吸引他们。
  3. 广告之间的PK（系统根据广告点击率、广告主预算等多种因素综合衡量），称之为竞价。

1.5 信息流广告常用术语。

1.5.1 互联网常用语

  1. PV：页面浏览量（评价网站流量的最常用指标之一）
  2. UV：独立访客数（和PV的区别在于去重，只记录一天内第一次进入网站的IP）
  3. DAU：日活（日活跃用户数，即每天使用产品的用户数，是一个用来评价产品用户量的指标）
  4. MAU：月活（月活跃用户数，即每月使用产品的用户数，也是反映产品用户量的指标）

1.5.2 媒体广告常用语

  1. 展示（曝光）：广告被一个用户看见一次称为一次展示（曝光）。广告主投放广告时，首先需要的就是曝光量，曝光量是广告投放最基础的数据指标。
  2. 点击：点击是曝光的下一步。点击是曝光的下一步，点击的发生可能是基于感兴趣，也可能是点错了。

第二章：广告投放流程

2.1 开户

获取广告账号的过程也叫开通账户，简称：开户。

2.1.2 开户的行业限制

  1. 法律规定： 法律规定一些行业和产品是不允许投放广告的。
  2. 平台的自身考量。平台出于对自身形象、用户体验上的考量，会对广告有进一步的限制。
  3. 行业默认：对媒体自身竞品不开户。

2.2 充值

推广账户的费用充值，不同平台要求不同。

2.3 广告内容审核常见的要求

经过媒体审核，确保投放内容合规。

  1. 不能出现虚假广告。
  2. 不能涉及色情、暴利的内容。
  3. 不能有保证性的承诺。
  4. 不能贬低别人。
  5. 不能侵犯版权。
  6. 不能用“国家级”、“最高级”、“最佳”等词语。
  7. 需要添加一些提示用语。

2.4 监测广告效果

2.4.1 数据监测的作用

通过技术手段，监测广告投放数据，用来确保广告投放数据的真实性和准确性。
数据统计是做一切数据分析的前提， 统计不到数据就什么都没有办法分析。
通过数据监测，我们能够精确统计到用户各个维度的行为，广告里的监测指标主要有曝光、点击、下载、激活、注册、下单、付费等行为。
完整的数据监测能够为后续跟踪广告投放效果、分析用户行为等工作打下基础。

2.4.2 数据监测方式

数据监测的方式和投放目的有关，广告中常见的投放目的有两种：增加app下载量和增加页面浏览量（投放落地页）。
投放app下载的时候， 有两种数据监测方式：一是用下载链接监测，而是用户专门的监测链接。一般认为专门的监测链接数据会更加准确一些，所以这种方式的使用频率更高。
增加页面浏览量，这个落地页链接本身也可以用来监测数据，此外也支持用专门的监测链接进行监测。
在数据监测里，优化师重点需要注意的是分渠道。不同创意，不同媒体区分不同的渠道。即区分不同来源的数据。这是做数据监测至关重要的一步。

2.4.3 常见数据监测公司

秒针系统，AdMaster（精硕科技），热云，TalkingData，友盟等。

2.5 广告的收费方式

2.5.1 广告拍卖会

投放还好的的收费方式主要有两种：媒体定价售卖和拍卖
传统电视台投放广告即为媒体定价售卖，价格相对较高
互联网广告的主流售卖方式就是拍卖。
- 最终成交价由市场竞争决定，而不是媒体单方面来定价，对广告主来说更公平。
- 对媒体来说实现了利益最大化，曝光始终是卖给此刻出价最高的人。

拍卖规则（不同点在于：用户行为不同）

  1. 按展示出价：每一次曝光最高愿意出多少钱。
  2. 按点击出价：每一次点击最高愿意出多少钱。
  3. 按转化出价：每一次转化最高愿意出多少钱。

用户漏斗行为： 曝光–点击–第一步转化–第二步转化–第三步转化。
转化是一个泛指词，点击之后的行为都可以叫做转化。具体什么叫做转化是广告主自己定义的。

选择不同的出价规则，媒体就会帮你优化什么。

  1. 按展示出价，媒体会帮你找到展示成本最低的用户，也就是降低cpm
  2. 按点击出价，媒体会帮你找到点击成本最低的用户，也就是降低cpc
  3. 按转化出价，媒体会帮你找到转化成本最低的用户，也就是降低转化成本。

2.5.2 计费点

常见计费点： 按点击付费 按曝光付费
主流计费方式：按转化出价但是是按展示或者点击计费。
按转化出价极大的降低了广告投放的风险。

2.5.3 “广义第二计费” 机制（又称第二价格密封拍卖）

媒体的计费是随着广告投放实时扣费。
第二价格密封拍卖：出价广告主不知道相互出价，媒体最终收取费用并不是按照获胜者自己的出价，而是按照下一名的出价再加一个很低的溢价（常见为0.01元），又称：广义第二计费
广义第二计费的特点

  1. 你的扣费不直接等于你的出价。
  2. 扣费是不确定的，因为你不知道下一名的出价是多少。
  3. 通常，实际扣费比出价略低一些。

广义第二计费的优势
“广义第二计费”机制被绝大多数竞价广告采用，能够避免价格大幅波动，还能够让广告主主动出更高的价格。如果按出价多少就扣多少，广告主就会在广告投出去后持续尝试降低出价（这样能少花钱），造成广告主们都在不停地出价，价格波动就会很大。广义第二计费机制的实际扣费取决于其他出价者，只要他们的出价不变，你调价格也没用（甚至可能因为调得过低而竞价失败，广告展现不出去）。广告主只用出自己能接受的最高价格就可以了

第三章：广告投放的基本原理

广告投放三部曲：选定人群–准备物料–广告竞价投放
投放广告的目的是为了和目标用户沟通，每一件产品都有自己的核心受众。

  1. 广告要重点投放给核心受众看，这就是投放广告的第一步： 选定广告投放人群，即解决“选定广告给谁看”的问题。
  2. 每一类用户都有相应的喜好，要用用户喜欢的方式来和他们沟通。这就是投放广告的第二步：准备广告投放的物料，即解决“给用户看什么”的问题。
  3. 与其他广告之间竞争，这就是投放广告的第三步：与其他广告主pk，也就是“广告之间的pk”。

3.2 定向—选定广告给谁看

产品一般会对自己的用户有一个用户画像，包括性别、年龄、几线城市等。
媒体提供人群划分维度供广告主选择，支持定位到用户的性别、地域和婚恋状况等。这是媒体的定向能力。
“选定广告给谁看”就是将这两者做匹配—用媒体支持的方式圈出自己想要的用户。
信息流广告也称效果广告，很大程度上是因为广告能够精准地投放给目标人群，大大节省广告费用。定向就是其中关键的一环。

3.2.1 产品的用户画像

每一个产品都有自己的定位，主要面向什么样的用户群，也就是用户画像。产品的用户画像一般可以从年龄、性别、地域和收入情况等几方面进行概括。

3.2.2 媒体支持的定向

定向的本质就是找人， 媒体帮助广告主找到投广告的人或排除哪些人。

媒体常见支持定向

  1. 用户基本情况：性别、年龄和地域。
  2. 兴趣：用户对什么感兴趣。
  3. 自定义人群：可以对人群做更灵活的处理，也支持广告主自己上传一些人群数据，一般称为人群包。

3.3 创意—给用户看什么

3.3.1 投放APP下载

创意（素材）指广告投放过程中用到的广告文案（或标题）、图片和视频。一般不包含落地页。

前端行为和后端行为
在信息流公告里，对用户行为有“前端”和“后端”之分。前端指媒体端，后端指广告主端。一般将媒体能直接收集到的行为算作前端行为，将媒体收集不到的行为算作后端行为。投放APP下载，媒体能收集到的最后一步行为是安装完成。常见的前端行为有展示、点击、下载和安装等。常见的后端行为有激活、注册和付费等。

3.3.2 投放落地页

广告主投放广告除了APP下载还有一个常见目的：获得销售线索或者直接卖东西。广告主花钱是为了让用户看到一个页面，这个页面统称为落地页。

无论投放app下载还是投放落地页，准备给用户看的内容都包含两部分：用户首先看到的“广告创意”和点击之后的“详细页面”。

3.4 竞价—广告之间的pk

在PK过程中，对广告主而言有两个关键指标—预算（打算花多少钱）和出价（能不能pk过别人，把钱花出去）

3.4.1 预算

准备花多少钱。（可实时调整）
预算是广告主投放广告的一个非常重要的指标。

3.4.2 出价

出价决定了广告的投放成本，出价越低那么成本也就越低。

3.4.3 媒体的流量分配逻辑

无论按展示出价、按点击出价还是按转化出价，媒体都是按照展示收费的，所以媒体关心的是每一个曝光能有多少收益，即：CPM
eCPM: 媒体对广告主出价的预估每个曝光的收益（平台通过广告主提供的产品名称、人群定向和广告创意预估人群是否转化，转化概率是多少，从而换算出cpm是多少，可以获取多少收益。

不同出价方式下eCPM 的计算方式

  1. 按展示出价：eCPM = CPM 出价， 不需要换算
  2. 按点击出价：eCPM = 点击出价 x 点击率 x 1000
  3. 按转化出价：eCPM = 转化出价 x 转化率 x 1000

在信息流广告中，常把点击率叫做CTR， 转化率叫做CVR，转化率可以指点击到转化，也可以指展示到转化，没一特殊说明的话， 一般指点击到转化，所以： eCPM = 转化出价 x CTR x CVR x 1000

转化出价可以等同于获取用户的成本，说明在投放广告时，可以通过优化广告创意、落地页和人群定向来提升点击率和转化率，进而提升eCPM，最终用比较低的成本获得用户。
