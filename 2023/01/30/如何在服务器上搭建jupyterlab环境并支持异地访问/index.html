如何在服务器上搭建jupyterlab环境并支持异地访问

安装jupyterlab

pip install –upgrade jupyter jupyterlab

生成JupyerLab配置文件

jupyter lab –generate-config
配置文件默认保存在: /home/UserName/.jupyter/jupyter_lab_config.py

设置jupyterlab登录密码

ipython3
from jupyter_server.auth import passwd
passwd()
输入密码后会得到一串哈希值,复制保留,稍等下有用.
exit() 退出ipython3界面

修改jupyterlab配置文件

vim /home/UserName/.jupyter/jupyter_lab_config.py
修改删除以下内容前的 # 号和空格

```
c.NotebookApp.allow_remote_access = True
c.NotebookApp.ip = ‘0.0.0.0’ # 允许任意IP段访问
c.NotebookApp.notebook_dir = ‘pwd’ # 设置文件默认保存位置
c.NotebookApp.open_browser = False # 设置运行时不启动浏览器
c.NotebookApp.password = u’sha1:**********’ # 设置前面生成的密码
c.NotebookApp.port = 8888 # 设置端口号
‘’’
