如何在服务器上搭建jupyterlab环境并支持异地访问

安装jupyterlab

生成JupyerLab配置文件

配置文件默认保存在: /home/UserName/.jupyter/jupyter_lab_config.py

设置jupyterlab登录密码

修改jupyterlab配置文件

修改删除以下内容前的 # 号和空格

设置中文环境

设置后台运行

screen -S jupyterlab # 开启一个名称为jupyterlab的会话
jupyterlab # 开启jupyterlab服务
# 快捷键 Ctrl + A + D 退出会话窗口, 服务仍然运行
screen -ls # 查看所有会话

设置开机自启

找到jupyterlab安装位置

which jupyter-lab

创建运行脚本

#!/bin/sh

/home/myjetson/.local/bin/jupyter-lab #（此处为jupyterlab安装位置，以上which jupyter-lab语句运行后内容）

修改运行脚本权限：

chmod 777 文件名

测试脚本正确性

./脚本文件名.sh

创建jupyterlab.service配置文件

sudo vim /etc/systemd/system/jupyter.service

粘贴以下内容到配置文件内

[Unit]
Description=Auto Load JupyterLab
After=network.target

[Service]
Type=simple
User=Ubuntu用户名
ExecStart=运行脚本位置/脚本文件名.sh
Restart=on-failure
RestartSec=15s

[Install]
WantedBy=multi-user.target

输入如下命令启动服务

sudo systemctl daemon-reload # 让systemd重新加载service文件
sudo systemctl enable jupyter.service # 设置开机自启

检查服务是否正常启动

sudo systemctl start jupyter.service
sudo systemctl status jupyter.service

如需关闭请输入以下命令

sudo systemctl stop jupyter.service
